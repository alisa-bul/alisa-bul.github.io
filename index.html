Обо мне

Я только начинаю свой путь в IT! Ранее работала аналитиком данных в сфере управления системой общего образования на региональном уровне (2 года, 9 месяцев). Имею опыт разработки систем показателей и методик для проведения мониторингов, обработки данных (Excel, Statistica), формирования аналитических отчетов, сборников и дашбордов в сжатые сроки.

Для изучения актуальных методов работы с данными я прошла курс "Аналитик данных" от школы KARPOV.COURSES. Сейчас самостоятельно продолжаю изучение:

  • Python
  • Библиотек для анализа данных и визуализации (Pandas, Numpy, SciPy, Statsmodels, Pingouin, Matplotlib, Plotly, Seaborn)
  • SQL
  • Инструментов для визуализации данных (DataLens, Power BI, Redash, Tableau)
  • Git
  • Apache Airflow
  • Статистика и A/B-тесты
  • Финансовые показатели и продуктовые метрики

Я открыта к изучению новой информации, готова выкладываться на все 100%. К решению рабочих задач подхожу с энтузиазмом и всегда стараюсь довести дело до конца. Меня не пугают рутинные задачи, ведь даже в них есть место творчеству.

Так, например, на предыдущем месте работы по результатам мониторинга для каждого муниципалитета региона ежегодно вручную формировались аналитические отчеты в Word (около 30 страниц). Это была монотонная работа, занимающая много времени. Я проявила инициативу и спроектировала дашборд, который значительно ускорил процесс. На выходе мы получили симпатичную и компактную визуализацию, наглядно демонстрирующую позитивные моменты и недочеты в работе муниципальных систем.

Также, благодаря совместным усилиям нашей команды в периоды проведения государственной итоговой аттестации (ГИА), готовности ответственных сотрудников работать в полную силу даже в выходные дни, наш региональный центр обработки информации два года подряд становился лидером в общероссийском рейтинге.

Учебный проект "Образовательная платформа"

Это мой финальный проект по результатам прохождения курса "Аналитик данных". В рамках проекта необходимо было выполнить три задания:

  • Провести A/B-тестирование новой механики оплаты услуг на сайте образовательной платформы, сформулировать выводы
  • Написать два SQL-запроса:
    • 1) информация о количестве очень усердных студентов;
    • 2) выгрузка таблицы с рассчитанными метриками для контрольной и тестовой групп: ARPU, ARPAU, CR в покупку, CR активного пользователя в покупку, CR пользователя из активности по математике в покупку курса по математике.
  • Написать функции в Python:
    • 1) пересчет метрик по результатам A/B-теста с учетом новых данных;
    • 2) построение графиков по получаемым метрикам.

Учебный проект по модулю "Продуктовая аналитика"

  • Использование фреймворка ICE для приоритизации гипотез
